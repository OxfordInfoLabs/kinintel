Example API Usage
List
Return the data for this datasource.

Filtering of the returned data can be effected by supplying parameters of the format FIELD_MATCHTYPE where FIELD matches one of the field names defined in the data source and MATCHTYPE should be one of the following values.

Match Type Description Example
eq for an equals match id_eq=3
neq for a not equals match id_neq=7
isnull for a null match id_isnull=1
isnotnull for a not null match id_isnotnull=1
gt for a greater than match id_gt=1
lt for a less than match id_lt=1
gte for a greater than or equals match id_gte=1
lte for a less than or equals match id_lte=10
startswith for a string starts with match name_startswith=app
endswith for a string ends with match name_endswith=app
contains for a string contains match name_contains=app
like for a wildcard or regular expression match. Simple wildcards can be supplied using * for one or more characters. Regular expressions can be supplied by surrounding the value with / delimiters. name_like=ap*, name_like=/^ap.*le$/
in for where the value is one of a list supplied in comma separated format name_in="apple,pear,banana"
notin for where the value is not one of a list supplied in comma separated format name_notin="apple,pear,banana"


Alternatively, advanced queries can be supplied using the built in Data Query Language passed as a query parameter which should be suitable URL encoded.

e.g. query=((id >= 5 && name like 'app*') || id < 3)

Results can be sorted by supplying a sort parameter which identifies columns to sort and directions (either asc or desc) using pipe separation

By default the first 100 rows will be returned. Paging can be achieved by supplying limit and offset parameters. 
curl -H "API-KEY:{{apiKeys ? apiKeys.apiKey : '[[apiKey]]'}}" -H "API-SECRET:{{apiKeys ? apiKeys.apiSecret : '[[apiSecret]]'}}"
 -X GET {{backendURL}}
                        /api/tabularData/{{datasourceUpdate.instanceImportKey}}?{{listQueryString}}
                        &limit=50&offset=10

Filtered Delete

                

                    Delete a set of rows using one or more filter expressions additively to identify the rows for
                    deletion for the data source identified by the passed import key.
                    


                    Filters are supplied as objects containing a column name supplied as the column property, a
                    match value supplied as the value property and a match type supplied as
                    the matchType property.
                    


                    Possible values for the matchType are:


                    

                        

                            eq
                            for an equals match
                        

                        

                            neq
                            for a not equals match
                        

                        

                            null
                            for a null match
                        

                        

                            notnull
                            for a not null match
                        

                        

                            gt
                            for a greater than match
                        

                        

                            lt
                            for a less than match
                        

                        

                            gte
                            for a greater than or equals match
                        

                        

                            lte
                            for a less than or equals match
                        

                        

                            startswith
                            for a string starts with match
                        

                        

                            endswith
                            for a string ends with match
                        

                        

                            contains
                            or a string contains match
                        

                    



                

                

                    
curl -H "API-KEY:{{apiKeys ? apiKeys.apiKey : '[[apiKey]]'}}" -H "API-SECRET:{{apiKeys ? apiKeys.apiSecret : '[[apiSecret]]'}}"

                    
 -X DELETE {{backendURL}}
                        /api/tabularData/filtered/{{datasourceUpdate.instanceImportKey}}
